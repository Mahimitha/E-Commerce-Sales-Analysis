# E-Commerce Sales Analysis

## Project Overview
This project analyzes the Olist Brazilian E-Commerce dataset to identify sales trends, top-performing product categories, regional performance, customer review patterns, and average order value trends.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Dataset
Olist Brazilian E-Commerce Dataset (Kaggle)

## Analysis Performed
1. Data Cleaning
2. Revenue Analysis by Product Category
3. Monthly Sales Trend Analysis
4. Regional Sales Performance
5. Average Order Value Trend
6. Customer Review Score Distribution

## Key Findings
- Highest Revenue Category: beleza_saude
- Peak Sales Month: November 2017
- Best Performing State: SP
- Highest Average Order Value: October 2016
- Majority of Customers gave 5-Star Reviews
